農業OS：次の進化ロードマップ

1. 管理作業ログ（施肥・防除）

経営改善・補助金・圃場分析の中心となる最重要領域。

● 必要なログ

  • 施肥：肥料種類・量・時期・担当者・コスト
  • 防除：農薬種類・希釈倍率・散布量・時期・担当者

● 圃場分析ページに追加するカード

  • 施肥履歴
  • 防除履歴
  • 施肥量 × 反収の散布図
  • 防除回数 × 反収の散布図

● 得られる価値

  • 補助金申請の強化
  • 肥料・農薬コストの最適化
  • 病害虫リスクの可視化
  • 反収改善の根拠が揃う

2. 育苗管理（播種 → 定植 → 収穫）

技術の見える化と、品種 × 圃場 × 育苗の相性分析の基盤。

● 必要なデータ構造

  • seedRef（品種・ロット）
  • trayCount（トレイ数）
  • sowingDate（播種日）
  • germinationRate（発芽率）
  • transplantDate（定植日）
  • transplantRef（どの plantingRef に紐づくか）

● リンク先

  • varieties-detail.json
  • plantingRef（圃場 × 品種 × 年度）
  • 収穫サマリー（反収）

● 圃場分析ページに追加するカード

  • 播種ログ
  • 育苗 → 定植 → 収穫の流れ
  • 育苗日数 × 反収の散布図

3. 品種サマリー分析（varieties-detail.json）

作付け計画の意思決定エンジン。

● 新規ファイル案

{
  "TEMPLATE_VARIETY": {
    "name": "",
    "type": "",
    "seedCost": "",
    "notes": ""
  },
  "品種A": {
    "name": "品種A",
    "type": "早生",
    "seedCost": 1200,
    "notes": "高温に強い"
  }
}

● 品種分析ページで見たい情報

  • 品種ごとの反収
  • 品種ごとの安定性（標準偏差）
  • 品種ごとの利益（反収 × 単価）
  • 品種 × 圃場の相性
  • 品種 × 面積の限界

● 散布図（品種版）

  • 横軸：作付面積
  • 縦軸：反収
  • 点：圃場
  • 色：品種
  • クリック：analysis/?field=◯◯

4. 散布図（面積 × 反収）

経営判断の地図になる可視化ツール。

● 圃場版

  • 横軸：作付面積
  • 縦軸：反収
  • 点：plantingRef
  • 色：品種
  • クリック：圃場分析へ

● 品種版

  • 横軸：面積
  • 縦軸：反収
  • 点：圃場
  • 色：品種

5. 天候・積算温度 × 収穫サマリー

生育予測の基礎データ。

● 必要なデータ

  • 日別気温
  • 積算温度
  • 降水量
  • 日照時間
  • 収穫日
  • 反収

● 分析できること

  • 積算温度と収穫日の関係
  • 積算温度と反収の関係
  • 気象条件と品種の相性
  • 異常気象時の反収低下パターン

6. NARO（農研機構）生育モデル

OS の AI 化の入口。

● できること

  • 生育ステージの予測
  • 病害虫リスクの予測
  • 収穫日の予測
  • 反収の予測

● OS に統合すると

  • 「あと◯日で収穫」
  • 「今週の施肥タイミング」
  • 「病害虫リスク：高」
  • 「予測収量：◯kg」

優先順位（収益 × 補助金の軸）

  1. 施肥・防除ログ（管理作業ログ）
  2. 育苗管理（播種 → 定植 → 収穫）
  3. 品種サマリー分析（varieties-detail.json）
  4. 天候・積算温度 × 収穫サマリー
  5. NARO 生育モデル
